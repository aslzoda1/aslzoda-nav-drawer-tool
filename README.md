# ✨ App Overview

**Smart Study Hub** is a modern productivity Android application designed to showcase practical Android development skills using real-world UI patterns and scalable architecture.

The project focuses on building a **clean, maintainable, and user-friendly application** with interactive UI components and offline-first storage.

---

# 🚀 Features

## 📋 Task Management System

✔ Add new tasks  
✔ Edit existing tasks  
✔ Delete tasks with confirmation  
✔ Undo delete using **Snackbar action**  
✔ Search and filter tasks easily  

---

## 🎨 Advanced UI Components

✔ Bottom Sheet for creating tasks  
✔ Custom Dialogs for editing and confirmation  
✔ Interactive Snackbars for user feedback  
✔ Smooth and responsive UI animations  

---

## 👤 User & Settings System

✔ Profile screen  
✔ 🌙 Dark Mode / ☀️ Light Mode toggle  
✔ Local user data persistence  
✔ Secure Log Out system  

---

## 💾 Offline Storage

✔ SharedPreferences based storage  
✔ JSON structured data storage  
✔ Fully **offline-first functionality**  
✔ Local JSON backup support  

---

# ⚡ Highlights

✨ Modern Android UI design  
✨ Modular fragment-based architecture  
✨ Interactive user experience  
✨ Offline-first data storage
